# Student-Portfolio
Responsive Student Portfolio Website using HTML, CSS &amp; JavaScript with projects, certificates, dark mode, and contact form.
Student Portfolio Website

This project is a fully responsive personal portfolio website developed using HTML5, CSS3, and Vanilla JavaScript as part of a web development assignment.

The website showcases my personal profile, educational background, technical skills, projects, and certificates in a clean and professional layout. It includes modern interactive features such as:

Responsive design for desktop, tablet, and mobile devices
Sticky navigation with smooth scrolling
Hero section with profile image
About Me and Education sections
Animated skill progress bars
Responsive project cards
Interactive certificate gallery with image preview modal
Dark/Light mode with local storage support
Contact form with JavaScript validation
CSS animations and hover effects

Technologies Used

HTML5
CSS3 (Flexbox & Grid)
JavaScript (ES6)
Git & GitHub

This project demonstrates the use of semantic HTML, responsive web design, DOM manipulation, and JavaScript-based interactivity without using any external frameworks.
